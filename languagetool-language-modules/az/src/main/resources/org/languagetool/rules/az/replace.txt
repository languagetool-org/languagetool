# form to the canonical AZ Latin spelling so LT can suggest the correct one.
#

# Only entries with at least one AZ-specific character are listed; pure ASCII
# words that don't need a diacritic fix are omitted.
#
# This is an intentionally conservative starter set (~300 high-frequency
# entries). Future PRs can extend it from larger normalization corpora.
