# Words to be marked as spelling errors even if the spell checker
# accepts them. Useful to improve the spell checker without touching
# its files (which are usually not from LT).
# Supported suffixes for German:
# /E will add e
# /S will add s
# /N will add n
# /A will add e, er, es, en, em (for adjectives - no comparative, no superlative)
# /F will add in (for gendered nouns)
# Lines ending with ".*" will prohibit all words starting with the previous string
# Lines starting with ".*" will prohibit all words ending with the following string
#  (this is identical to the regular expression;
#   therefore, the suffixes /E, /S, etc. will not work for these special cases)
# For the opposite (accept words), see ignore.txt.
