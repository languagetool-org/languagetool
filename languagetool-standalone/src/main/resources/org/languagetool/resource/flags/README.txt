The flags icon set downloaded from the Open Icon Library
and downscaled with the command
convert -background none -resize 16x16 iso/svg/flag-[ISO 3166 2-letter code].svg [ISO 3166 2-letter code].png

Original README.txt
########################################################
#           International Flags Package                #
########################################################

Package source:	
	Open Icon Library
	http://openiconlibrary.sourceforge.net/

Date:		
	16 Dec 2009

License:	
	Public Domain

Original Source of svg file:	
	Wikimedia Commons - Category:SVG sovereign state flags
	http://commons.wikimedia.org/wiki/Category:SVG_sovereign_state_flags

Description: 
	This package include flags for most independent and dependent 
nations.  This includes svg and png images of many sizes and shapes.  
The original directory includes images with the shape of the actual 
flags.  Standard and iso directories include square images with the 
flags.  The flags still have the actual shape with the excess space 
filled with transparency.  Original and standard directories have 
files named with the nations names.  The iso directory has files 
named with the ISO-3166 codes.

ISO-3166 Codes: 
	http://www.iso.org/iso/english_country_names_and_code_elements
